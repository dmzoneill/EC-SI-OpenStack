EC-SI-OpenStack
===============

Configure the top of the script with your network settings

Requirements:
    
Controller (can also be compute):
    
    Vmware VM (or physical) with:
    
        3 HDD (sda os 20gb, sdb block 20gb, sdc object 10gb)
        2 nics physical or virtual
        
Compute:
    
    Vmware VM (or physical) with:
    
        3 HDD (sda os 20gb, sdb block 20gb, sdc object 10gb)
        2 nics physical or virtual
